## Adding Your Organization

We'd love to list you as an adopter! This helps:
- Build community confidence
- Connect you with other users
- Influence roadmap priorities

### Requirements

- Active use of AgentMesh (production or serious evaluation)
- Willingness to be listed publicly
- Optional: Brief description of use case

### How to Add

1. **Easy:** [Open an issue](https://github.com/imran-siddique/agent-mesh/issues/new?template=adopter.yml) with your details
2. **PR:** Edit this file directly and submit a pull request

### What We'll List

- Organization name
- General use case (no confidential details needed)
- Optional: Contact person (GitHub handle)
- Optional: Logo (with permission)

### Confidential Adopters

If you're using AgentMesh but can't be listed publicly, we can still:
- Count you in our adopter metrics (anonymously)
- Include you in foundation applications (with NDA)
- Provide priority support

Contact: adopters@agentmesh.dev
